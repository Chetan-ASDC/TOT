# Combined Task – Day 1 (Foundation Excel Practice)

---

## Objective
This combined task is designed to **revise and apply all Day 1 concepts** in one continuous, practical exercise.  
Students will practice **data entry, formatting, formulas, logical functions, text functions, and basic analysis** without using any Day 2 topics.

---

## Scenario
You are working as an **Office Executive**.  
You are asked to prepare and maintain an **Employee Salary Sheet** using Excel with proper formatting and calculations.

---

## Task 1: Data Entry & Sheet Setup

Create a worksheet with the following columns:

- Employee ID  
- Employee Name  
- Department  
- Date of Joining  
- Basic Salary  

### Instructions
- Enter at least **10 employee records**
- Use proper file name and worksheet name
- Adjust column width appropriately

---

## Task 2: Formatting

Apply the following formatting:

- Make headings **bold**
- Center-align headings
- Apply **Date format** to Date of Joining
- Apply **Currency format** to Salary
- Apply borders to the data range
- Freeze the header row

---

## Task 3: Basic Formulas

Below the salary column, calculate:

- Total Salary using `SUM`
- Average Salary using `AVERAGE`
- Highest Salary using `MAX`
- Lowest Salary using `MIN`

---

## Task 4: Logical Function (IF)

Add a new column called **Salary Status**.

### Condition:
- If Salary ≥ 30,000 → “Eligible”
- Else → “Not Eligible”

Use the `IF` function.

---

## Task 5: Text Functions

Perform the following operations:

1. Clean Employee Names:
   - Remove extra spaces using `TRIM`
   - Convert names to proper case using `PROPER`

2. Create a new column:
   - Extract **first name** from Employee Name using:
     - `LEFT`, `MID`, `RIGHT`, or `FIND`

---

---

## Task 6: Basic Data Analysis 

- Count total number of employees using `COUNT` or `COUNTA`
- Identify how many employees are:
  - Eligible
  - Not Eligible

---



## Final Submission Requirements

Students must submit:
- One properly formatted Excel file
- Clean and readable data
- Correct formulas and functions applied
- Ready-to-print worksheet
