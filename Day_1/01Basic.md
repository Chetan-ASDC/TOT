## Excel Absolute Basics


## 1. What is Excel

### Concept
Excel is a **spreadsheet application** used to:
- Store data in rows and columns  
- Perform calculations  
- Analyze and summarize data  

Excel is mainly used for **data handling and decision making**.

### Trainer Explanation
Explain Excel as:
- A smart table  
- A calculator that can handle large data and logic  

### Demo
- Open a blank Excel workbook  
- Type numbers in different cells  
- Show how Excel auto-adjusts rows and columns

## 2. Workbook vs Worksheet

### Concept
- **Workbook**: The Excel file (example: `Sales.xlsx`)  
- **Worksheet**: Individual sheets inside a workbook  

### Explanation Points
- One workbook can have multiple worksheets  
- Worksheets are used to organize different datasets  

### Demo
- Add, rename, and delete worksheets  
- Switch between sheets  

---

## 3. Rows, Columns, and Cells

### Concept
- **Rows** run horizontally (1, 2, 3…)  
- **Columns** run vertically (A, B, C…)  
- **Cell** is the intersection of a row and a column  

### Key Explanation
- Each cell has a **unique address**  
- Example: Column A + Row 1 = **A1**

### Demo
- Click different cells  
- Highlight row numbers and column letters  

---

## 4. Cell Address

### Concept
A **cell address** tells Excel exactly where data is located.

### Examples
- A1 → Column A, Row 1  
- C5 → Column C, Row 5  

### Practice Task
Ask learners to identify:
- Cell B3  
- Cell D7  
- Cell A10

## 6. Cell Referencing

### Types of Cell References

#### Relative Reference
- Changes when copied  
- Example: `=A1+B1`  

#### Absolute Reference
- Does not change when copied  
- Example: `=$A$1+$B$1`  

#### Mixed Reference
- Partially fixed  
- Examples:  
  - `$A1`  
  - `A$1`  

### Demo
- Copy a formula across cells  
- Show how results change with each reference type
